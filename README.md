# 📅 Cal3x | Your Calendar at a Glance

A lightweight macOS menu bar utility that tracks your schedule and updates its icon based on your real‑time status.

## 🕒 Features
* **Smart Indicators:** Icons shift from 🕒 to 🚀 to 🔴 automatically.
* **Privacy First:** Direct Google API connection. No middle-man servers.
* **Quick Actions:** Add events and tasks in seconds using natural language.

## 🛠️ Installation
1. Download the `dmg` file above.
2. Drag the file to the `/Applications` folder.
3. Launch **Cal3x**.
4. Log in with your Google account.
